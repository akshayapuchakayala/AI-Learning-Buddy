# 🎓 AI Learning Buddy

An AI-powered Learning Buddy built using **Streamlit** and **Google Gemini AI**. This application helps beginners learn any topic by providing simple explanations, real-life examples, quizzes, and answers to user questions through an interactive web interface.

## 🚀 Live Demo

https://ai-learning-buddy-hpuwix5nedrg7lxnioaeue.streamlit.app/

> **Note:** If the AI response is temporarily unavailable, it may be due to Google Gemini API quota limits. Please try again later.

---

## 📌 Features

- 📖 Explain any topic in simple language
- 🌍 Provide real-life examples
- 📝 Generate 5 multiple-choice quiz questions with answers
- 💬 Ask anything related to the topic
- 🎯 Beginner-friendly interface
- 🌐 Web application built using Streamlit

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/akshayapuchakayala/AI-Learning-Buddy.git
```

Go to the project folder:

```bash
cd AI-Learning-Buddy
```

Install the required packages:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```

---

## 🔑 API Configuration

Create a Streamlit Secret named:

```
GEMINI_API_KEY
```

Add your Google Gemini API key as the value.

Example:

```toml
GEMINI_API_KEY="YOUR_GEMINI_API_KEY"
```
